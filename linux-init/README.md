#Linux系统安装初始化脚本
##1. CentOS 7.x 系列
```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/wl1524520/linux-scripts/master/linux-init/centos7.sh)"
```
或者
```bash
sh -c "$(wget https://raw.githubusercontent.com/wl1524520/linux-scripts/master/linux-init/centos7.sh -O -)"
```
